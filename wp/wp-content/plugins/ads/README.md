# Ads Board — Плагин доски объявлений для WordPress

Ads Board — это плагин для WordPress, который добавляет функционал доски объявлений на ваш сайт. Позволяет создавать, редактировать и публиковать объявления с изображениями, ценами, контактами и категориями.

## Основные возможности
✅ Создание и редактирование объявлений через админ-панель
✅ Категории объявлений с иерархией и сортировкой
✅ Загрузка нескольких изображений с установкой главного
✅ Фильтрация и сортировка объявлений (по дате, цене, статусу)
✅ Закреплённые и важные объявления (приоритет в выдаче)
✅ Проверка срока действия объявлений (авто-снятие с публикации)
✅ Адаптивный фронтенд с сеткой карточек
✅ Страница отдельного объявления с галереей и контактами
✅ Шорткод [latest_ads] для вставки объявлений в любые записи
✅ Настройки плагина: количество на странице, формат даты, отображение элементов
✅ Подсчёт просмотров с защитой от накрутки

## Установка и запуск

### Требования
- WordPress 6.0+
- PHP 8.0+
- MySQL 5.7+

### Ручная установка
1. Скачайте папку плагина ads
2. Загрузите её в директорию /wp-content/plugins/
3. Активируйте плагин в админ-панели: Плагины → Установленные плагины

## Как пользоваться

### Для администратора
1. Доступ к панели управления
- После активации в меню админки появится пункт Объявления:
- Все объявления — список, фильтрация, массовые действия
- Добавить объявление — форма создания/редактирования
- Категории — управление категориями
- Настройки — общие параметры плагина
2. Создание объявления
- Заполните обязательные поля: заголовок, описание, автор, категория
- Укажите хотя бы один контакт: телефон или email
- Загрузите изображения (первое станет главным)
- Настройте дату публикации и окончания (опционально)
- Установите статус: «Черновик», «Опубликовано», «Продано»
3. Управление категориями
- Создавайте, редактируйте и удаляйте категории
- Задавайте порядок сортировки (число: чем меньше — тем выше)
- Просматривайте количество объявлений в каждой категории
4. Настройки плагина
- Количество объявлений на странице (1–100)
- Формат отображения даты
- Показывать/скрывать просмотры, автора
- Количество колонок в сетке
- Размер изображений в превью
- Включение модерации новых объявлений

### Для пользователя
| Страница         | URL                     | Описание                                                  |
|------------------|-------------------------|-----------------------------------------------------------|
| Главная доска    | /board/                 | Список всех активных объявлений с фильтрами и сортировкой |
| Категория        | /board/category/{slug}/ | Объявления конкретной категории                           |
| Список категорий | /board/category/        | Все категории с количеством объявлений                    |
| Объявление       | /board/ad/{slug}/       | Детальная страница с галереей, описанием, контактами      |

### Шорткоды
Вставляйте объявления в любые записи или страницы:
```text
[latest_ads]                          # 6 последних объявлений
[latest_ads count="10"]               # 10 объявлений
[latest_ads category="real-estate"]   # Только категория "Недвижимость" (по slug)
[latest_ads category="3"]             # Только категория с ID=3
[latest_ads sort="price_desc"]        # Сортировка: цена по убыванию
[latest_ads show_price="false"]       # Скрыть цену в карточках
[latest_ads columns="2"]              # 2 колонки в сетке
[latest_ads class="my-custom-block"]  # Доп. класс для обёртки
```

## Структура базы данных
Плагин создаёт следующие таблицы (с префиксом wp_ по умолчанию):
| Таблица           | Описание                         |
|-------------------|----------------------------------|
| wp_ads            | Основные данные объявлений       |
| wp_ads_categories | Категории объявлений             |
| wp_ads_images     | Изображения объявлений (галерея) |
| wp_ads_settings   | Настройки плагина                |

## Безопасность
- Все пользовательские данные экранируются через esc_html(), esc_attr(), esc_url()
- SQL-запросы выполняются через $wpdb->prepare() с защитой от инъекций
- Все формы защищены через wp_nonce_field() / wp_verify_nonce()
- Загружаемые файлы проверяются по MIME-типу и расширению
- Доступ к админ-функциям ограничен через current_user_can('manage_options')
